# bb-visual-production

Production is where strategy either holds or collapses. Every asset is a brand touchpoint.

---

## Core Principle

Visual production is not execution — it's interpretation. Every production decision either reinforces or dilutes the brand system. Speed without standards is brand erosion.

---

## Critical Reflex: Before Production Starts

| Check | If missing... |
|---|---|
| Brand identity system finalized? | Production will be inconsistent |
| Clear brief with defined output specs? | Production will be revised endlessly |
| Audience and context defined? | Can't make the right visual choices |
| Approved brand assets available? | Stop. Get assets first. |

A production brief without these four things will cost twice as long to deliver.

---

## Production Brief Standard

Every production job needs a brief — even a simple social post:

```
ASSET TYPE      — What is being produced
FORMAT          — Dimensions, file format, resolution
CONTEXT         — Where will this appear and how will it be seen
MESSAGE         — One primary communication
VISUAL DIRECTION — Reference to brand system + any specific direction
DEADLINE        — Hard delivery date
DELIVERABLES    — Exact file list expected
```

No brief = no start.

---

## Quality Checklist

Before any asset leaves production:

- [ ] Correct dimensions for intended platform/medium
- [ ] Typography follows brand system
- [ ] Color values match brand tokens (not approximations)
- [ ] Logo usage follows brand guidelines
- [ ] Copy is final and approved
- [ ] Safe zones respected (text/logo not cropped on any platform)
- [ ] File exported at correct resolution/format
- [ ] Named and organized correctly for handoff

---

## Platform Specs Reference

| Platform | Primary format | Key constraint |
|---|---|---|
| Instagram feed | 1080×1080 / 1080×1350 | Visual impact in small size |
| Instagram story | 1080×1920 | Safe zone: top/bottom 250px |
| LinkedIn | 1200×627 | Professional tone |
| Twitter/X | 1200×675 | High contrast text |
| Print A4 | 2480×3508px at 300dpi | Bleed 3mm |

Always verify current platform specs — they change.

---

## Animation Brief Standard

For any motion work:
```
PURPOSE         — Why does this need to move?
DURATION        — Target length
STYLE           — Reference to brand motion principles
KEY MOMENT      — The one frame that must land
SOUND           — Yes/no, music or SFX direction
OUTPUT          — File format, loop/no loop, resolution
TOOL            — Rive / Cavalry / After Effects / Spline
```
